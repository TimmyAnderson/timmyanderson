IT BOOKS


This page contains some of IT books I have studied during my 20+ years career as a programmer. When studying IT books I always try to create useful code samples that serve as a cookbook when I need to use or understand some feature. All of these code samples are my personal work, not the downloaded samples from associated book web pages. The comments and remarks are written in my native language Slovak. All source codes are available on the GITHUB.

Advanced .NET Debugging

Author: Mario Hewardt
Publisher: Addison-Wesley Professional
Publication date: 9 November 2009
Pages: 542
Review: A quite interesting book about advanced debugging techniques in .NET that go far beyond 'standard' debugging tools in Visual Studio. Normally you don't need to use these techniques but in some cases post mortem debugging, doing COM interoperability, or having problems with memory (Large Object Heap is not defragmented in .NET), techniques presented in this book are being very helpful.

Applications = Code + Markup: A Guide to the Microsoft Windows Presentation Foundation

Author: Charles Petzold
Publisher: Microsoft Press
Publication date: 13 September 2006
Pages: 1020
Review: The book from legendary author Charles Petzold well known for his Win32 API programming book about WPF. The book provides unique way for explanation of WPF. First half of the book is all about C# code. No XAML at all. It shows you WPF programming model that is independent to XAML. The second part of book is all about XMAL. It teaches you how to create static and dynamic forms only in XAML. Since WPF is a part of .NET Core, it's an evolving technology that allows you to create non-web GUI application under Windows, Linux or MacOS.

C++ Standard Library, The: A Tutorial and Reference, 2nd Edition

Author: Nicolai M. Josuttis
Publisher: Addison-Wesley Professional
Publication date: 30 March 2012
Pages: 1136
Review: C++ Standard Library covers virtually all aspects of the C++ Standard Library for C++ 11. The book contains hundreds of examples that are easy to follow. Strongly recommended to every C++ developer.
Link: Source codes

CLR via C#, 4th Edition

Author: Jeffrey Richter
Publisher: Microsoft Press
Publication date: 15 November 2012
Pages: 896
Review: CLR via C#, 4th Edition is by far the most comprehensive and deepest explanation how C# and CLR works. It gives you a lot of insights how the CLR works. There is also a large section about multithreading and task programming with one of the best explanation of how thread, tasks and synchronization primitives work. Jeffrey Richter as a seasoned expert in Windows technologies provides a unique, clear and easy to understand the CLR and C#. It is a must read for every C# developer. The book was written for .NET Framework but it is perfectly valid for .NET Core. I consider CLR via C# as the best book that has been written about the CLR and C#.

Compiler Construction: Principles and Practice

Author: Kenneth C. Louden
Publisher: Cengage Learning
Publication date: 24 January 1997
Pages: 592
Review: Excellent introduction into construction of compilers. The books describes frontend and backend part of compilers like scanners, expression trees, parsers, semantic analyzers, runtime environments and code optimization. Using solely this book I was able to implement my own trivial programming language in pure C# without using tools like YACC or BISON.

Author: Steven Pratschner
Publisher: Microsoft Press
Publication date: 26 February 2005

Entity Framework Core in Action, Second Edition, 2nd Edition

Author: Jon P Smith
Publisher: Manning
Publication date: 8 June 2021
Pages: 624
Review: The best book about Entity Framework in .NET Core. The book gives a lot of details about the many aspects of EF which is essential tool for DB access in .NET Core. Essential COM

Author: Don Box
Publisher: Addison-Wesley Professional
Publication date: 22 December 1997
Pages: 464
Review: A nice introduction in Microsoft COM technology. Even though parts of this technology are outdated like DCOM or COM+, COM objects are still essential part of Windows system. And this book gives you a good introduction into 'raw' COM, not using high level frameworks like MFC or ATL, but only a raw C++.

Expert .NET 2.0 IL Assembler

Author: Serge Lidin
Publisher: Springer
Publication date: 24 August 2006
Pages: 529
Review: Normally you don't need to know about MSIL and still being a successful .NET programmer. MSIL is underlying assembler language to which .NET languages like C# compile its code to. I believe that most .NET programmers never ever write any code in MSIL since C# and other languages are must easier to use. But if you like to delve deep into technology and how it works this book does a good job. CLR also offers a few features that are not supported in C# and can be accessed only via MSIL. A newer edition of this book is available.

Inside Windows Communication Foundation

Author: Justin Smith
Publisher: Microsoft Press
Publication date: 16 May 2007
Pages: 304
Review: Inside Windows Communication Foundation shows you a lot of internals of WCF including creating your own communication channels. Now WCF is obsolete technology supported only for .NET Framework but if you really want to see what it is under the hood of WCF, this book is you. Using this book I was able to create my own trivial communication channel in WCF.

JavaScript: The Definitive Guide: Activate Your Web Pages, 6th Edition

Author: David Flanagan
Publisher: O'Reilly Media
Publication date: 13 May 2011
Pages: 1096
Review: I read this book out of curiosity. I don't use (and don't like) JavaScript in my projects. But even now when there are many alternatives how to create Web frontends like strongly typed compiled TypeScript or WebAssembly, JavaScript is still de-facto a standard of frontend web development. And David Flanagan's book is the most comprehensive and the best learning tool of this language. Strongly recommended for every web frontend developer. By the way, 7th edition of this book is available.

Programming the Microsoft Windows Driver Model, 2nd Edition

Author: Walter Oney
Publisher: Microsoft Press
Publication date: 26 December 2002
Pages: 880
Review: A very interesting book about Windows kernel programming. I'm not a kernel programmer, but this book is still very useful, since it gives you a lot of insides how Windows kernel works and how different is from user mode programming. Windows Driver Model (WDM) is a very complex technology and Microsoft created Windows Driver Foundation to simplify Windows driver development. But if you look to see what is under the hood of Windows OS just out of curiosity, this book is for you.
I haven't finished the book yet. Many chapters are still missing.
