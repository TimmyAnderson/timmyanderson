IT BOOKS

The page contains some of IT books I have studied during my 20+ years career as a programmer. When studying IT books I always try to create useful code samples that serve as a cookbook when I need to use or understand some feature. All of these code samples are my personal work, not the downloaded samples from associated book web pages. The comments and remarks are written in my native language Slovak. All source codes are available on the GITHUB.

C++ Standard Library, The: A Tutorial and Reference, 2nd Edition

Author: Nicolai M. Josuttis
Publisher: Addison-Wesley Professional
Publication date: 30 March 2012
Pages: 1136
Review: C++ Standard Library covers virtually all aspects of the C++ Standard Library for C++ 11. The book contains hundreds of examples that are easy to follow. Strongly recommended to every C++ developer.
Link: Source codes

CLR via C#, 4th Edition

Author: Jeffrey Richter
Publisher: Microsoft Press
Publication date: 15 November 2012
Pages: 896
Review: CLR via C#, 4th Edition is by far the most comprehensive and deepest explanation how C# and CLR works. It gives you a lot of insights how the CLR works. There is also a large section about multithreading and task programming with one of the best explanation of how thread, tasks and synchronization primitives work. Jeffrey Richter as a seasoned expert in Windows technologies provides a unique, clear and easy to understand the CLR and C#. It is a must read for every C# developer. The book was written for .NET Framework but it is perfectly valid for .NET Core. I consider CLR via C# as the best book that has been written about the CLR and C#.
Link: Source codes

Compiler Construction: Principles and Practice

Author: Kenneth C. Louden
Publisher: Cengage Learning
Publication date: 24 January 1997
Pages: 592
Review: Excellent introduction into construction of compilers. The books describies frontend and backend part of compilers like scaners, expression trees, parsers, semantic analyzers, runtime environments and code optimization. Using solely this book I was able to implement my own trivial programming language in pure C# without using tools like YACC or BISON.

Essential COM

Author: Don Box
Publisher: Addison-Wesley Professional
Publication date: 22 December 1997
Pages: 464
Review: A nice introduction in Microsoft COM technology. Even though parts of this technology is outdated like DCOM or COM+, COM objects are still essential part of Windows system. And this book gives you a good introduction into 'raw' COM, not using high level frameworks like MFC or ATL, but only a raw C++.
Link: Source codes

JavaScript: The Definitive Guide: Activate Your Web Pages, 6th Edition

Author: David Flanagan
Publisher: O'Reilly Media
Publication date: 13 May 2011
Pages: 1096
Review: I read this book out of curiosity. I don't use (and don't like) JavaScript in my projects. But even now when there are many alternatives how to create Web frontends like stongly typed compiled TypeScript or WebAssembly, JavaScript is still de-facto a standard of frontend web development. And David Flanagan's book is the most comprehensive and the best learning tool of this language. Strongly recommended for every web frontend developer. By the way, 7th edition of this book is available.

Author: Joe Duffy
Publisher: Addison-Wesley Professional
Publication date: 28 October 2008

Joe Duffy an expert in concurrent programming doesn't teach you only about concurrent programming techniques on native Win32 platform and .NET plafrom but gives you a lot of insides about parallelism and concurrent programming problems in general.

Windows via C/C++, 5th Edition

Authors: Jeffrey M. Richter, Christophe Nasarre
Publisher: Microsoft Press
Publication date: 28 November 2007
Pages: 848
Review: Windows via C/C++ is the best book about Win32 API library. It gives you a deep dive on such complex topics like threading, thread synchronization, IO operations using Completion Ports, virtual memory, structured exception handling and many other topics. Jeffrey Richter's uniqe writing style makes even the most complex topics easy to understand.
